Contextual Quick Buy — UX Spec

Especificación visual (estática, estilo Figma) del feature Contextual Quick Buy para el storefront B2B de LvlUp WholeSale. Sirve para analizar el flujo, la anatomía de componentes y los estados antes de codificar. No es funcional.

Tema oscuro con tokens confirmados vía retrieve del branding set B2B_Commerce: fondo #11132C, superficie #1C2049, CTA verde, acento violeta, texto blanco e inputs blancos.
